# Required Packages

* python 2.7
* git
* ipython notebook
* scikit-learn >= 0.15.0
* pandas >= 0.14.1
* matplotlib >= 1.1.1rc2

## Debian-based (Ubuntu, Mint, CrunchBang) Installation

```apt-get install python git python-pip ipython ipython-notebook  
pip install -U scikit-learn pandas matplotlib```


## OS X Installation

`pip install -U scikit-learn pandas matplotlib`
